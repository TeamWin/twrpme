Man twrp command line 

install 	install zip file. 
wipe 	wipe PARTITION: 
	PARTITON are, one by line: 
		cache, 
		system, 
		dalvik, 
		data.
backup 	PARTITIONS-TO-BACKUP [BACKUP-NAME]: backup partition PARTITIONS-TO-BACKUP to [BACKUP-NAME]
	PARTITIONS-TO-BACKUP, multiples selection allowed:
		S: system, 
		D: data, 
		C: cache, 
		R: recovery, 
		B: boot, 
		A: and-sec, 
		E: sd-ext, 
		M: skip md5 creation.
	[BACKUP-NAME] If not provided, current date/time is used. 

restore  	PARTITION-TO-RESTORE BACKUP-NAME: restore partitions from existant backup: 
	PARTITION-TO-RESTORE, multiples selection allowed: 
		S: system, 
		D: data, 
		C: cache, 
		R: recovery, 
		B: boot, 
		A: and-sec, 
		E: sd-ext, 
		M: skip md5 check. 
	[BACKUP-NAME]: could be full path or name of twrp backup in current twrp backup folder. 
	Backup couldn't be encrypted.
